# 🔐 NetScan

NetScan is a Python-based network scanner and vulnerability analysis tool.

## 🚀 Features
- Port scanning
- OS detection
- Hostname resolving
- Vulnerability analysis
- Report generation

## 🛠️ Technologies Used
- Python
- Socket Programming
- Cybersecurity Concepts
